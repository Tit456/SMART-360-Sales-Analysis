# SMART-360-Sales-Analysis

## 📌 Project Overview
SMART-MART 360 is a data analytics project that analyzes supermarket sales data across multiple branches (Ikeja, Lekki, Surulere). The goal is to generate actionable insights on revenue, customer behavior, product performance, and business trends using Excel for data cleaning and Power BI for visualization.

---

## 🎯 Objectives
- Analyze total revenue, transactions, and profit performance
- Identify top-performing products and categories
- Evaluate customer segments and payment methods
- Track revenue trends by time (month, day, quarter)
- Provide business insights for decision-making

---

## 🛠️ Tools & Technologies
- Microsoft Excel (Data Cleaning & Preparation)
- Power BI (Dashboard Development & Visualization)
- DAX (Data Analysis Expressions)

## 🧹 Data Cleaning Process (Excel)
- Removed duplicates and null values
- Standardized column names and formats

---

## 📊 Dashboard Features (Power BI)
- KPI Cards (Total Revenue, Transactions, Gross Profit)
- Revenue breakdown by:
  - Day of the week
  - Month and Quarter
- Customer segmentation (Wholesaler, Regular, Walk-in)
- Payment method analysis (POS, Transfer, Cash)
- Top-performing products and categories
- Branch filtering (Ikeja, Lekki, Surulere)

---

## 📈 Key Insights
- Lekki branch generates the highest revenue overall
- December records the peak monthly sales
- Saturdays drive the highest daily revenue
- Food & Beverages is the top-performing category
- Electronics accessories yield the highest margins
- Discounts above 5% significantly reduce profitability
- Some products require urgent restocking due to high demand


## 🚀 How to Use
1. Download the `.pbix` file from the `dashboard` folder
2. Open using Power BI Desktop
3. Interact with filters and visuals to explore insights
